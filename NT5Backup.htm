PE Builder v3 plugin

Windows XP Backup

This pebuilder plugin requires:

1)  The following files from an XP System:   ntbackup.exe, ntmaspi.dll, vssapi.dll

2)Copy the files from your Windows\System32 directory.

Work In Progress!!!! This plugin does work, but with some errors
Here's how it works:

Launch Windows XP Backup and you will get a couple of errors, all are passable if you follow these directions:

1. First error:
 
Click OK

I like to do things manually, so I usually turn off the wizard and then choose "Let me choose what to back up.

At the end, when you're about to hit Finish, select Advanced

Ensure that you select to disable Volume Shadow Copy.

Backup will backup to a file now on your removable media. I recommend using a network share or removable drive.
